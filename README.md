### TheBlog Website

A full-stack blogging website like Medium. Check it out:
https://blog-website-two-chi.vercel.app/

## Features
- User Authentication
- Create Blogs
- Read Blogs
- Update Blogs
- Delete Blogs
- Follow Users
- Pagination

## Tech Stack

- Typescript
- Hono
- Prisma
- PostgreSQL
- React
- Tailwind
